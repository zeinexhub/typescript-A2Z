### Why should I use TypeScript ?

- TypeScript helps make code clearer by defining what type of data is used.

> TypeScript ব্যবহার করলে কোডে কোন ডেটা কোন টাইপের তা আগে থেকেই স্পষ্ট থাকে।

- It shows errors before the code runs if the wrong data type is used.

> এতে ভুল টাইপের ডেটা ব্যবহার করলে কোড চালানোর আগেই error দেখায়।

- This makes TypeScript safer and easier to work with than JavaScript.

> JavaScript যেখানে এসব ভুল ধরতে পারে না, TypeScript সেখানে কোডকে বেশি safe এবং সহজবোধ্য করে।
