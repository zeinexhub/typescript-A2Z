### String

- Represents text data. Can use single quotes ('), double quotes ("), or backticks (`) for template literals.

> এটি টেক্সট ডেটা প্রকাশ করে। String লেখার জন্য single quotes ('), double quotes ("), অথবা backticks (`) ব্যবহার করা যায় template literals এর জন্য।

```ts
let color: string = "blue";
let fullName: string = "John Doe";
let age: number = 30;
let sentence: string = `Hello, my name is ${fullName} and I'll be ${
  age + 1
} next year.`;
```
