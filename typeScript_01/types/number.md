### Number

- Represents both integers and floating-point numbers. TypeScript uses the same number type for all numeric values.

> এটি integers এবং floating-point numbers উভয়কে প্রকাশ করে। TypeScript সব ধরনের সংখ্যার জন্য একই number টাইপ ব্যবহার করে।

```ts
let decimal: number = 6;
let hex: number = 0xf00d;
let binary: number = 0b1010;
let octal: number = 0o744;
let float: number = 3.14;
```
