### Introduction

- Typescript is an object oriented and strongly typed programming language which is a superset of JS designed by MICROSOFT. TypeScript is JS with some additional features.

> TypeScript হলো MICROSOFT কর্তৃক তৈরি একটি object-oriented ও strongly typed প্রোগ্রামিং ভাষা, যা JavaScript-এর superset। TypeScript মূলত JavaScript, তবে এতে কিছু অতিরিক্ত ফিচার যোগ করা হয়েছে।

- Why TypeScript : a. Compilation b. Provides an optional system for JS code c. Strong Static typing d. Supports object oriented programming e. Supports type definitions <br> Typescript supports other JS libraries, it is portable, it is superset of JS, it is static, oop

> TypeScript কেন ব্যবহার করবো : a. Compilation – কোড চালানোর আগে JavaScript-এ রূপান্তর করা হয়। b. Optional type system – JavaScript কোডে চাইলে type ব্যবহার করা যায়। c. Strong static typing – কোড রান হওয়ার আগেই type-এর ভুল ধরা পড়ে। d. Object-oriented programming support – class, interface ইত্যাদি সাপোর্ট করে। e. Type definitions support – ডেটার structure আগে থেকেই নির্ধারণ করা যায়। <br> TypeScript অন্য JavaScript লাইব্রেরিগুলো সাপোর্ট করে, এটি portable অর্থাৎ বিভিন্ন প্ল্যাটফর্মে ব্যবহার করা যায়, এটি JavaScript-এর superset, এতে static typing আছে এবং এটি object-oriented programming সাপোর্ট করে।

- oop language, supports static typing, Errors can be found and corrected during compile time, support for ES3, ES4, ES5, ES6, function can have optional parameters, converted into JS code to be understandable for browsers, proper build setup is required for static definitions

> এটি একটি OOP ভাষা, যেখানে static typing সাপোর্ট করে। কোড চালানোর আগেই (compile time) ভুল খুঁজে বের করে ঠিক করা যায়। এটি ES3, ES4, ES5, ES6 সাপোর্ট করে এবং function-এ optional parameter ব্যবহার করা যায়। TypeScript কোডকে JavaScript-এ রূপান্তর করা হয় যেন ব্রাউজার বুঝতে পারে। Static definition ব্যবহারের জন্য সঠিক build setup প্রয়োজন হয়।

- typescript can be used to develop js applications for both client side and server side execution. It can be deployed instead of JS language since it adds more features and gives error directly with the code.Typescript is used for building large-scale applications for enterprises.

> TypeScript ব্যবহার করে client side এবং server side—দুই জায়গারই JavaScript অ্যাপ্লিকেশন তৈরি করা যায়। এতে অতিরিক্ত ফিচার থাকায় JavaScript-এর পরিবর্তে এটি ব্যবহার করা যায় এবং কোডের মধ্যেই সরাসরি error দেখায়। TypeScript সাধারণত enterprise-level বড় আকারের অ্যাপ্লিকেশন তৈরির জন্য ব্যবহার করা হয়।

- advantages : Highlights errors at compilation time during the time of developement, it runs on any browser or JS engine. it has a namespace concept by defining a module. supports strongly typed or static typing.

> ডেভেলপমেন্টের সময়ই (compile time) error গুলো দেখায়। এটি যেকোনো browser বা JavaScript engine-এ রান করতে পারে। Module define করার মাধ্যমে namespace concept সাপোর্ট করে। এতে strongly typed / static typing সাপোর্ট রয়েছে।

- disadvantages : typescript takes a long time to compile the code. it does not support abstaract classes. extra compilation step for converting typescript into JS. overly complicated typing system.

> TypeScript কোড compile করতে বেশি সময় লাগে। এটি abstract class সাপোর্ট করে না। TypeScript থেকে JavaScript-এ রূপান্তরের জন্য অতিরিক্ত compilation step প্রয়োজন হয়। এর typing system তুলনামূলকভাবে জটিল, যা নতুনদের জন্য কঠিন হতে পারে।
