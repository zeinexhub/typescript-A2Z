### What is TypeScript ?

- TypeScript is a programming language built on JavaScript that adds static typing.

> TypeScript হলো JavaScript-এর উপর ভিত্তি করে তৈরি একটি প্রোগ্রামিং ভাষা, যেখানে static typing যোগ করা হয়েছে।

- It uses the same syntax as JavaScript but allows developers to define types for safer code.

> এটি JavaScript-এর একই syntax ব্যবহার করে, কিন্তু নিরাপদ কোড লেখার জন্য type নির্ধারণ করার সুবিধা দেয়।
