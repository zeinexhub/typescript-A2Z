### BigInt

- Represents whole numbers larger than 2⁵³ - 1. Use the n suffix to create a bigint.

> এটি পূর্ণ সংখ্যা প্রকাশ করে যা 2⁵³ - 1 এর চেয়ে বড়। BigInt তৈরি করার জন্য সংখ্যার শেষে n ব্যবহার করা হয়।

```ts
const bigNumber: bigint = 9007199254740991n;
const hugeNumber = BigInt(9007199254740991); // Alternative
```
