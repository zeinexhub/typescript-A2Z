### Boolean

- Represents true/false values.Used for flags, toggles, and conditions.

> এর অর্থ হলো true বা false মান প্রকাশ করা। এটি সাধারণত flag, toggle এবং condition বোঝানোর জন্য ব্যবহার করা হয়।

```ts
let isActive: boolean = true;
let hasPermission = false;

//TypeScript নিজে থেকেই বুঝে নেয় যে কোনো ভেরিয়েবল বা ভ্যালু true/false টাইপ, তাই আলাদা করে boolean লিখে দিতে হয় না।
```
